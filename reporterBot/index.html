Report Bot

Prefix: `ps `

Functionality:

I made this bot for people to report their feedback, questions/suggestions or bugs they found in any of my applications.

Commands:

Help

Guild only: no
Admin only: no
Cooldown: 0

Gives you a list of all the commands.

Guild only: no
Admin only: no
Cooldown: 0

Gives you all the information (for example their usage) about a specific command.

Report

Guild only: yes
Admin only: no
Cooldown: 60

I made this command for you guys to send in your feedback, questions/suggestions or bugs.
Once used, the bot will automatically send you a DM and ask you to specify your report. When you fill this in correctly it will generate a report and send it in the bug report channel in my server.

Reload

Guild only: no
Admin only: yes
Cooldown: 0

I use this command to update command files without having to turn the bot off and on again.

Delete

Guild only: yes
Admin only: yes
Cooldown: 0

Deletes all messages by a user that starts with the given input.
Mostly used to clear messages from the bot.

Announce

Guild only: yes
Admin only: yes
Cooldown: 0

Announces a new version of one of my projects to everyone who has the @Announcement role.
